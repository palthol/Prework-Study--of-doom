HTML

  • HTMl stands for Hypertext Markup language!
  • HTML is the standard language used to create and design webpages
  • HTML provides the structure of a website by using a series of elements composed of tags, content and attributes which define the layout of content and the functionality of the website
  • A HTML document is split into the head and the body.
    • the head provides essential information about the website.
    • The body contains all other visible content that the end user can interact with.

CSS

  • CSS stands for cascading style sheets!
  • CSS is a language that is used to change the appearance and format of HTML or XML documents.

Git

  • Git is a distributed version control system!
  • Git was designed to help developers track changes in their code to collaborate on projects more efficiently.

Javascript

  • JavaScript is a high level programming langauge!
  • JavaScript is a very versatile programming language and can perform a large variety of tasks.
  • In this context we are utilizing javaScript in unison with HTML and CSS to create a cohesive and interactive website for our end user.
